# Netflix-Clone

Cloning Netflix Home and Landing User Interface (UI) to practice ReactJs, React Router, Responsive UI, and Authentication with MongoDB

# REQUIREMENT

In Order to run this repository localy to your local machine. You need several preparation.

- Installed NodeJs
- MongoDB Cluster Account

# GET STARTED

1. Clone this repository to your working directory
2. Open a new terminal and run `cd backend` to get current directory inside backend folder
3. run `npm install` to install all the depedencies required by backend
4. run `npm start` to start backend server live
5. Open a new terminal seperated from backend and run `npm install` to install all the depedencies required by frontend
6. run `npm start` to start frontend live to your localhost

## !IMPORTANT

In Order to run this website, Backend server, MongoDB, and Frontend must live at the same time.
